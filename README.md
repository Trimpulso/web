# Web: Hola mundo

Este repositorio contiene una página mínima que muestra "Hola mundo".

- Archivo principal: `index.html`
- Despliegue: GitHub Pages desde la rama `main`

## Visualización
Una vez habilitado GitHub Pages, la página estará disponible en:

```
https://trimpulso.github.io/web/
```

## Desarrollo local
Abre `index.html` directamente en tu navegador o sirve la carpeta con cualquier servidor estático.

```sh
# ejemplo con Python
python3 -m http.server 8080
# luego visita http://localhost:8080
```
